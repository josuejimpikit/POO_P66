README - Simulación de Robots (MVC)

Cambios realizados:
- Se agregó un atributo de escudo fijo (30%) que reduce el daño recibido.
- Se incluyó una habilidad especial (doble ataque) que puede usarse una vez por robot.
- Se implementó un torneo "todos contra todos", asegurando que los robots no se ataquen a sí mismos.
- Se añadió un reporte final de batalla que muestra: estado final, energía, daño infligido y recibido.
